Informe de Comprobantes internos de cta. cte. Este informe permite obtener una lista de los Comprobantes internos de cuenta corriente registrados en el sistema en un período de tiempo determinado. El listado sólo considerará los comprobantes internos cuyas fechas estén dentro del rango indicado. También posible seleccionar los comprobantes a listar por su prefijo y número. El informe incluye, por cada comprobante, el dato de la fecha, tipo, prefijo, número, código y nombre del cliente y, si las hubiera, las observaciones del comprobante. Es posible, además, incluir en la impresión del informe una carátula en la que se detallan todas las opciones seleccionadas por el usuario al solicitarlo. Para emitir el listado primero se ingresan los datos en las distintas fichas contenidas en la tarea, de manera tal de filtrar la información de acuerdo a las necesidades. Luego se debe presionar el ícono (botón) de la barra de herramientas superior que corresponda al tipo de emisión requerida. El uso de estos botones se detalla en el link Emisión disponible en un párrafo posterior. Al momento de imprimir un informe (tanto por pantalla como por papel) se puede utilizar el formato standard del sistema, o indicar un formato personalizado de impresión. Este último se indica en el campo homónimo que, en general, se encuentra en la zona inferior de la primer solapa y su funcionalidad está detallada a continuación en el link Emisión. Emisión FCC (Fecha de certeza contable) Los datos a ingresar para solicitar la emisión del listado son: Ficha Rangos Período a considerar Prefijo Prefijo correspondiente a los comprobantes internos de cuenta corriente que se desean incluir en el listado. El prefijo debe corresponder a un talonario que admita el ingreso de este tipo de comprobantes. Si este dato es dejado en blanco se incluirán en el listado los comprobantes internos de cuenta corriente sin tener en cuenta su numeración. Número Números de los comprobantes internos de cuenta corriente que se desean incluir en el listado. Puede ingresarse un número en particular o un rango de números. Si este dato es dejado en blanco y se indicó un prefijo, se incluirán todos los comprobantes internos de cuenta corriente del prefijo indicado. Imputación contable Códigos de las imputaciones contables de clientes que se desean incluir en el listado. Puede seleccionarse una cuenta en particular o un grupo de cuentas. Si este dato es dejado en blanco, se incluirán en el listado, todas las cuentas asociadas a los clientes. Si se indica alguna imputación sólo se listará para cada cliente los comprobantes vinculados con las imputaciones que se indicaron. Mostrar estado actual de las aplicaciones Este campo permite indicar si el listado detallará las aplicaciones actuales o las iniciales. Vale aclarar que el estado de este campo no afecta el detalle de las aplicaciones correspondientes al origen ya que éstas no se pueden modificar, por lo que para el origen siempre se detallarán las aplicaciones que se cargaron durante ingreso del comprobante. En cambio, para las aplicaciones del destino, cuando se indique que se muestre el estado actual de las aplicaciones, se detallarán las aplicaciones actuales que son las que se observan a través del proceso de aplicaciones de cuenta corriente para el destino, caso contrario, se detallarán las aplicaciones cargadas durante el ingreso del comprobante para el destino. Por defecto, se detallarán las aplicaciones iniciales. Ordenamiento Indicación del ordenamiento que se le desea dar al listado. Las opciones son: Cliente Comprobante Si se selecciona ordenamiento por Cliente los datos listados se podrán ordenar por código o por nombre del cliente, dependiendo de la opción que se elija entre las ofrecidas seguidamente. Código Nombre Si se selecciona ordenamiento por Comprobante se mostrarán los datos listados ordenados por talonario, prefijo y número de comprobante. Ficha Clientes Empresas //
