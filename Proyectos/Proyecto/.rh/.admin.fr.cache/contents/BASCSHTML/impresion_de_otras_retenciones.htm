Otras retenciones Esta tarea permite imprimir comprobantes de retención definidos como otros tipos de retenciones, correspondientes a órdenes de pago ya ingresadas al sistema. El número del comprobante de retención, el formato de impresión, la impresora y la cantidad de copias a imprimir están indicados en la ficha Retenciones de la Configuración de Empresas. Para realizar la tarea se debe proceder de acuerdo a lo que se indica en Impresión de comprobantes de retención.
