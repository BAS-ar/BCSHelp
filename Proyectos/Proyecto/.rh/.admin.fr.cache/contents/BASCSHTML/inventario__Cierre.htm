Cierre de inventario Mediante esta tarea se cierra un inventario físico en un depósito determinado. Cada inventario se inicia en un depósito, luego se registran los bienes que lo componen con sus cantidades y mediante esta tarea concluye la toma de ese inventario, el mismo queda cerrado y el depósito queda nuevamente habilitado para la entrada y salida de mercadería. Se puede cerrar un inventario mientras se está tomando inventario en otros depósitos. Se debe indicar el depósito y en ese momento se muestra el número de inventario que se cierra y la fecha de cierre del mismo. Los datos a ingresar para cerrar un inventario son:   Depósito Número de depósito en el cual se cierra el inventario. Debe corresponder a un depósito definido en el sistema y con un inventario iniciado. Al indicar el depósito, el sistema muestra el número de inventario que se va a cerrar y la fecha de inicio del mismo.
