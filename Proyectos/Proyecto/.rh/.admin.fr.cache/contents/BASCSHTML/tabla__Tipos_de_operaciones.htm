Tipos de operaciones Esta tabla permite definir tipos de operaciones a utilizan para categorizar las distintas operaciones, o sea, los comprobantes que se ingresan en el sistema. Este dato permite identificar comprobantes para darles un tratamiento diferencial en diferentes procesos o informes del sistema. Código Código  de hasta tres caracteres. Descripción Denominación del tipo de operación. Como ejemplo, se muestra como se informa en un comprobante de compras.
