Reapertura Esta tarea permite realizar la reapertura de una orden de producción que ya está cerrada, tanto si fue registrada en una planta como en un depósito. No pueden reabrirse órdenes de producción cerradas si los subdiarios de las partes de producción asociados ya están cerrados a esa fecha.   Número Se debe indicar el talonario del comprobante asociado a la orden de producción y el número de aquella que se desea reabrir. Fecha Al indicar el dato en el campo anteriormente mencionado (Número), el sistema exhibe aquí la fecha de la orden de producción que se desea cerrar.   Planta Se exhibe el código de planta relacionada con la orden de producción que se reabre.   Depósito En este campo se exhibe el código del depósito destino de la producción.   Hoja de ruta El sistema exhibe el número de Hoja de ruta asociada a la orden de producción que se desea reabrir.   Cantidad En este campo el sistema exhibe la cantidad de bienes producidos relacionado con la orden de producción que se reabre.
