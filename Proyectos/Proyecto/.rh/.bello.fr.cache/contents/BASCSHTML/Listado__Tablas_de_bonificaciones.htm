Tablas de bonificaciones A través de esta tarea es posible emitir un informe que detalle los datos de las distintas tablas de bonificaciones definidas en el sistema. Las tablas de bonificaciones se definen para un atributo de clientes y un atributo de ítems y permiten establecer los porcentajes de bonificación que se aplicará para cada combinación de valores de estos dos atributos. Además, es posible aplicar automáticamente distintos porcentajes en función de la cantidad de unidades vendidas. Se pueden listar las tablas vigentes a una fecha determinada o las tablas con una fecha de vigencia específica. En el primer caso, si no se indica una fecha ni una tabla, el listado muestra todas las tablas para todas las vigencias ingresadas. En el segundo caso siempre se debe indicar una fecha y si no se indica una tabla, el listado muestra todas las tablas vigentes a esa fecha. Para cada tabla de bonificaciones que se incluye en el informe se muestran todos los valores de atributos de clientes y de ítems con la cantidad de unidades y el porcentaje de bonificación para cada cantidad y para cada combinación de valores de atributos. Es posible, además, incluir en la impresión del informe una carátula en la que se detallan todas las opciones seleccionadas por el usuario al solicitarlo. Los datos que deben ingresarse para emitir el listado son:   Tabla Códigos de las tablas de bonificaciones que se desean incluir en el listado. Puede seleccionarse una tabla en particular o un grupo de tablas. Si este dato es dejado en blanco, se incluirán todas las tablas en el listado, ya sea las que tienen una fecha de vigencia determinada o las vigentes a la fecha indicada. Opciones Método de selección de las tablas de bonificaciones a incluir en el informe. Día de vigencia Si se selecciona Día de vigencia se listarán todas las tablas que tengan una fecha de vigencia igual a la fecha indicada. Vigente al Si se selecciona Vigente al se listarán todas las tablas que tengan estén vigentes a la fecha indicada. Fecha En ambos casos se debe indicar luego la fecha para la cual listan las tablas. Para emitir el listado primero se ingresan los datos en la tarea, de manera tal de filtrar la información de acuerdo a las necesidades. Luego se debe presionar el ícono (botón) de la barra de herramientas superior que corresponda al tipo de emisión requerida. El uso de estos botones se detalla en el link Emisión disponible en un párrafo posterior. Al momento de imprimir un informe (tanto por pantalla como por papel) se puede utilizar el formato standard del sistema, o indicar un formato personalizado de impresión. Este último se indica en el campo homónimo que se encuentra en la zona inferior de la pantalla y su funcionalidad está detallada a continuación en el link Emisión.   Emisión
