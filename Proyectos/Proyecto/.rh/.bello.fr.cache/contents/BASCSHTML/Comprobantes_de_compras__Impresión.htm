Impresión Permite imprimir los comprobantes registrados en el sistema. La cantidad de copias, formato de impresión, tipo de emisión (impresora o archivo) se definen en el talonario correspondiente. Independientemente de lo que se indique en el talonario, se dispone de botones para emitir el comprobante por impresora y en un archivo PDF. El formato de impresión está contenido en un archivo de extensión QRP que contiene los datos del comprobante que se deseen imprimir, la ubicación de los mismos en el formulario, la forma que tiene el encabezado, cuerpo y pie del formulario, etc. Si se imprime en archivo, si se desea, se podrá utilizar una plantilla a utilizar como fondo. Los datos a ingresar para la emisión son:   Tipo de comprobante Tipo de comprobante de compras que se desean imprimir. Las opciones son: Comprobante interno (compras) Factura de compra A Factura de compra B Factura de compra C Factura de compra A Factura de compra I Factura-Recibo de Compra A Factura-Recibo de Compra B Factura-Recibo de Compra C Liquidación A (compras) Liquidación B (compras) Liquidación C (compras) Liquidación I (compras) Liquidación Créd. A (compras) Liquidación Créd. B (compras) Liquidación Créd. C (compras) Liquidación Créd. I (compras) Nota de Crédito A (compras) Nota de Crédito B (compras) Nota de Crédito C (compras) Nota de Crédito I (compras) Nota de Débito A (compras) Nota de Débito B (compras) Nota de Débito C (compras) Nota de Débito I (compras) Ticket (compra) Proveedor Código del proveedor titular de él o los comprobantes que se desea imprimir. Si este dato es dejado en blanco, el sistema permitirá imprimir los comprobantes de todos los proveedores sin distinción. Desde número Número de comprobante de compra desde el que se desea imprimir. Deben completarse el prefijo y el número. Si el número no se completa, se realizará la impresión desde el primer comprobante de compra existente para el prefijo indicado. El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección. El prefijo trae asociado el número de talonario de comprobantes de compra a imprimir. Hasta número Número de comprobante de compra hasta la que se desea imprimir. Deben completarse el prefijo y el número. Si el número no se completa, se realizará la impresión hasta el último comprobante de compra existente para el prefijo indicado. El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección. Si se indica desde y hasta un mismo número de comprobante, se realizará la emisión de ese comprobante compra en particular. Sólo los comprobantes no impresos Indicación de si se desean emitir solamente los comprobantes de compra que aún no han sido emitidos o todos los que se incluyen en el rango de números indicado.   Período a considerar Para tener en cuenta El ingreso de rango de fechas no es obligatorio. El sistema al momento de imprimir verifica si para el rango de números indicado hay comprobantes repetidos en más de una vigencia, avisando al usuario de tal eventualidad sugiriendo indicar un rango de fechas para evitar las repeticiones. Además, se consulta al usuario si desea seguir con la impresión o no. En caso de continuar con la impresión el sistema imprimirá todos los comprobantes en el rango indicado teniendo en cuenta el rango de fechas indicado aunque haya números de comprobante repetidos, por lo que un mismo número de comprobante podría imprimirse más de una vez para diferentes vigencias de fechas.
