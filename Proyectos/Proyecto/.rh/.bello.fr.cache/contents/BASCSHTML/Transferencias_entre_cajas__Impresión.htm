Impresión de transferencias Permite imprimir los comprobantes registrados en el sistema. La cantidad de copias, formato de impresión, tipo de emisión (impresora o archivo) se definen en el talonario correspondiente. Independientemente de lo que se indique en el talonario, se dispone de botones para emitir el comprobante por impresora y en un archivo PDF. El formato de impresión está contenido en un archivo de extensión QRP que contiene los datos del comprobante que se deseen imprimir, la ubicación de los mismos en el formulario, la forma que tiene el encabezado, cuerpo y pie del formulario, etc. Si se imprime en archivo, si se desea, se podrá utilizar una plantilla a utilizar como fondo. Los datos a ingresar para la emisión son:   Desde número Número de comprobante de la transferencia desde el que se desea imprimir. Deben completarse el prefijo y el número de comprobante. Si el número no se completa, se realizará la impresión desde la primer transferencia existente para el prefijo indicado. El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección. El prefijo trae asociado el número de talonario de las transferencias a imprimir. Hasta número Número de comprobante de la transferencia hasta el que se desea imprimir. Deben completarse el prefijo y el número de comprobante. Si el número no se completa, se realizará la impresión hasta la última transferencia existente para el prefijo indicado. El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección. Si se indica desde y hasta un mismo número de comprobante, se realizará la emisión de esa transferencia en particular.
