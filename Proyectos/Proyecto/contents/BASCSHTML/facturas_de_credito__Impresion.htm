Impresión de facturas de crédito

Desde esta tarea es posible imprimir facturas de crédito generadas (FC).

Pueden imprimirse las FC seleccionándolas a través de un talonario de facturas de venta y un rango de números correspondiente al mismo, o pueden imprimirse las FC generadas por una factura de venta en particular.

También pueden seleccionarse las FC a imprimir a través del talonario de FC y un rango de números correspondiente al mismo, o pueden imprimirse determinadas FC.

La cantidad de copias a imprimir de cada FC depende del talonario que se indique. La impresora por la cual se emiten las FC y el formato de impresión de las mismas también vienen indicados en el talonario.

El formato de impresión está contenido en un archivo de extensión QRP que contiene los datos del comprobante que se deseen imprimir, la ubicación de los mismos en el formulario, la forma que tiene el encabezado, cuerpo y pie del formulario, etc.

El sistema se entrega con un archivo denominado SPBFFC01.QRP que contiene el formato de impresión standard de las facturas de crédito. Este archivo puede ser modificado por el usuario según las necesidades de la empresa.

Los datos a ingresar para la emisión son:

 

Facturas de venta

Si se indica esta opción se podrán seleccionar las FC a imprimir mediante la selección de la o las facturas de venta que le dieron origen.

Talonario

Número de talonario correspondiente a las facturas de venta de las cuales se desea imprimir las FC.

Debe corresponder a un talonario que permita el ingreso de comprobantes de venta y que tenga asociado un talonario de FC. Este último tendrá informada la cantidad de copias a imprimir, la impresora y el formato de impresión de las FC.

Desde número

Número de la factura desde las cual se desea imprimir las FC.

Deben completarse el prefijo y el número. Si el número no se completa, se realizará la impresión desde el primer comprobante de venta existente con el prefijo indicado.

El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección.

Hasta número

Número de la factura hasta la cual se desea imprimir las FC.

Deben completarse el prefijo y el número. Si el número no se completa, se realizará la impresión hasta el último comprobante de venta existente con el prefijo indicado.

El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección.

Si se indica desde y hasta un mismo número de comprobante, se realizará la emisión de todas las FC generadas a partir de ese comprobante de venta en particular.

 

Facturas de crédito

Talonario

Número de talonario del cual se desean imprimir las FC.

Debe corresponder a un talonario que permita el ingreso de FC y el mismo traerá asociado la cantidad de copias a imprimir, la impresora y el formato de impresión de los comprobantes.

Desde número

Número del comprobante desde el que se desea imprimir.

Deben completarse el prefijo y el número. Si el número no se completa, se realizará la impresión desde la primer FC existente con el prefijo indicado.

El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de FC generadas que provee el sistema en este punto para permitir la selección.

Hasta número

Número del comprobante hasta el que se desea imprimir.

Deben completarse el prefijo y el número. Si el número no se completa, se realizará la impresión hasta la última FC existente con el prefijo indicado.

El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de FC generadas que provee el sistema en este punto para permitir la selección.

Si se indica desde y hasta un mismo número de comprobante, se realizará la emisión de esa FC en particular.
