Importación de exenciones impositivas

Formato

Formato del archivo que se desea importar.

Es posible indicar que un mismo registro se ingrese como exención de percepciones y de retenciones al mismo tiempo. Esto es útil en los casos en que un mismo CUIT se asocia a un cliente y a un proveedor y se quiere ingresar para ambos un mismo porcentaje de exención.

Para ello, en el campo Tipo del formato, se dispone de los siguientes valores:

A Para indicar que el registro se ingresa como exención de percepción y de retención de Ingresos Brutos.

C Para indicar que el registro se ingresa como exención de percepción y de retención de Ganancias.

Y Para indicar que el registro se ingresa como exención de percepción y de retención de IVA.

Si se indica alguno de estos valores para un CUIT que no se repite en clientes y proveedores, el registro se ingresará como exención de percepción si el CUIT existe en clientes, y como la exención de retención que corresponda, si el CUIT corresponde a proveedores.

También debe tenerse en cuenta que un CUIT puede pertenecer a la empresa en cuyo caso las exenciones que se importen se importarán como exenciones de la empresa.

Nombre archivo Examinar

Se debe indicar el nombre del archivo a importar, puede buscarse mediante el botón Examinar.

Preproceso

Archivo

Es posible indicar un archivo para preproceso del archivo a importar, con extensión awk, que se aplicará automáticamente sobre el archivo a importar antes de iniciar el proceso de importación. Se dispone de un botón para buscar dicho archivo. Para que este proceso funcione es preciso que el archivo awk.exe se ubique en directorio Programs de la instalación de BASCS. Vale aclarar que AWK es un lenguaje de programación diseñado para procesar datos basados en texto. Esta funcionalidad puede utilizarse en los casos en que el archivo a importar tiene un formato muy especial y debe adecuarse antes de la importación.

Es posible no eliminar, al finalizar el proceso de importación, los archivos intermedios generados al utilizar un archivo de preproceso, o en la eliminación de caracteres extraños del archivo original.

Para ello podrá definirse una variable ambiental de usuario con nombre NOBORRARARCIMP, que en caso de crearse indicando que tiene como valor 1, provoca que el sistema no elimine los archivos mencionados al finalizar el proceso de importación.

Los archivos se podrán encontrar en el directorio de instalación de BASCS o en la ubicación del archivo de importación. El correspondiente a la aplicación del archivo de preproceso tiene el prefijo ~PRE y la extensión TXT. El correspondiente a la eliminación de caracteres extraños tiene el mismo nombre que el archivo de importación y la extensión BAS.

En caso de no existir la variable o no tener valor 1, dichos archivos se eliminarán al finalizar el proceso de importación, tal como ocurría hasta ahora.

Solo reemplazar los registros que hayan cambiado

TTildar si sólo se desea reemplazar los registros que hayan cambiado.

Eliminar coeficientes anteriores a meses

Es posible informar que se eliminen coeficientes anteriores, para evitar resguardar innecesariamente la información de las exenciones de Ingresos Brutos de meses anteriores.

En caso que no se indique una cantidad de meses o dicha cantidad sea igual a cero, no se eliminarán coeficientes aunque los mismos estén vencidos. 
Si se indica una cantidad de meses el sistema procederá a la eliminación de porcentajes de exención cuya fecha de vencimiento sea menor a la cantidad de meses indicada respecto de la fecha actual.

Valores por defecto

% Percepción

Porcentaje que se utilizará para las percepciones si el archivo que se importa tiene indicado valor cero o no tiene indicado nada para alguno de los registros.

% Retención

Porcentaje que se utilizará para las retenciones si el archivo que se importa tiene indicado valor cero o no tiene indicado nada para alguno de los registros.

Provincia

Código de la provincia que se utilizará para las retenciones o percepciones de ingresos brutos si el archivo que se importa no tiene indicada la provincia o no tiene indicado nada para alguno de los registros.

Generar

Para comenzar la importación se debe presionar el botón Generar.

Para tener en cuenta

Al finalizar el proceso de importación, el proceso informa la cantidad de registros importados, la cantidad de registros eliminados y los registros que no se hayan podido importar por presentar alguna anomalía, ya sea por exenciones que se superponen, números de CUIT incorrectos u otros problemas.
