Cierre de camión

El Cierre del camión se realiza luego de cerrar una o mas guías, de registrar la entrega de los valores que cobró el conductor en el reparto y de descargar la mercadería del camión.

Los datos que se muestran son:

Camión

Código del camión que se desea cerrar.

Se debe ingresar el código de un camión activo que tenga asociados una caja y un depósito.

Luego de ingresado el camión se mostrarán la caja y el depósito del mismo y los datos del último cierre efectuado para el camión, con fecha de cierre y número de lote.

Informes de cierre

Existen dos que permiten acceder a los siguientes listados:

Arqueo de Caja

Detalle de movimientos

Para un correcto se verifica que el usuario tenga permisos para acceder a estos listados, de lo contrario se muestra un aviso que informará tal situación.

Si se dispone de , los informes se precargan con los datos correspondientes al camión (depósito, caja y lote).
