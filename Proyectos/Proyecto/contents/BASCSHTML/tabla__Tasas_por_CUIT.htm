Tasas por CUIT

Esta tabla permite ingresar manualmente las tasas de percepción y retención de ingresos brutos correspondientes a un determinado padrón.

Los valores también pueden importarse desde un archivo haciendo uso del utilitario BCSIMPIB (Importación del padrón para ingresos brutos).

Para cada padrón es posible ingresar diferentes períodos de validez de las tasas ingresadas para retención y percepción, así como la fecha de publicación de las mismas.

Si se utiliza la API BasPadrones, se muestran los porcentajes de BasPadrones actualizados al momento de indicar el padrón y el CUIT que se desea consultar o modificar.

Debe tenerse en cuenta que si un porcentaje se modifica en forma manual desde este programa, y luego se importan los padrones desde la API BasPadrones, no se sobreescribirán esos porcentajes modificados en forma manual desde este programa.

Padrón

Código del padrón para la cual se desea informar las tasas de percepción y retención.

El padrón debe haberse dado de alta previamente en la tabla Padrón de tasas.

CUIT

CUIT de la empresa para la cual se ingresa la información.
