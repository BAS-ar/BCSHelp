Parámetros de autorización de Listas de precios 

Se puede implementar el sistema de modo que las listas de precios requieran de autorización para poder ser utilizadas. El que la requieran o no y, en el caso que la requieran, las condiciones que hacen que una lista de precios deba autorizarse, quedan a elección del usuario.

Es posible indicar una tabla de autorizaciones para todas las listas de precios en general y/o indicar otras tablas para las listas de precios que incluyen determinados bienes.

Los parámetros pueden modificarse mientras se está operando con el sistema y la actualización de las opciones que se indiquen se realizará en forma inmediata sin necesidad de salir del sistema y volver a entrar.

Los parámetros de listas de precios posibles de configurar son:

Para todas las listas de precios

Indicación de si se desea que todas las listas de precios se autoricen. La forma en que deberá efectuarse la autorización dependerá de la tabla de autorizaciones que se indique.

Tabla de autorización

Tabla que mediante la cual se desea autorizar

Con ítems determinados

Indicación de si la autorización de listas de precios dependerá de los bienes que incluyan.

Atributo

Código de atributo que se desea utilizar para definir que bienes deben incluir las listas que han de autorizarse. Debe corresponder a un atributo ya definido en el sistema y aplicable a Items.

Esta opción sólo se habilita si se indicó la opción Con Items determinados.

Tabla de valores

En esta tabla se deberán ingresar los distintos valores del atributo de ítems, indicado en el campo precedente, para los cuales se desea definir un circuito de autorización.

Valor

Valor del atributo cuyo código se indicó en el campo Atributo.

Esta opción sólo se habilita si se indicó la opción Con items determinados.

Las listas de precios que incluyan ítems que tengan este valor para el atributo indicado en el campo correspondiente, requerirán autorización para poder ser utilizadas en los comprobantes y la forma en que debe efectuarse la autorización dependerá de la tabla de autorizaciones que se indique.
