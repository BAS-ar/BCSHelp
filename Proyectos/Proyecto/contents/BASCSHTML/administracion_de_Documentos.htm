Administración de Documentos

Permite adjuntar archivos externos. Una vez grabado el comprobante la administración de los archivos adjuntados se realiza desde el Proceso de Adjuntos.

 Tablas/ Configuración/ Parámetros/ Administración de documentos.

La información de la grilla no es editable, y se administra a través de los botones Agregar, Eliminar y Comentarios, ubicados a la derecha de la misma.

Se pueden adjuntar documentos alojados en cualquier directorio, siempre que la PC desde la cual se estén adjuntando tenga acceso al directorio en cuestión. Vale aclarar que los archivos a adjuntar no deben estar siendo utilizados por ningún usuario.

Es posible adjuntar archivos de cualquier formato (.zip, .pdf, .doc, .exe, etc.) y no existe restricción en la cantidad de archivos a adjuntar.

Una vez adjuntado el archivo, si se hace clic sobre él, el sistema intenta abrirlo con el programa que corresponda a la extensión del mismo (si por ejemplo es .doc, lo abrirá con Word, si es xls con Excel, etc.).

Si por algún motivo no puede abrirse, se ofrece una ventana para que el usuario elija un programa con el cual abrir el documento en cuestión.

Agregar

Oprimiendo este botón, se abre una ventana en la cual se debe especificar la ruta en la cual se encuentra el archivo a adjuntar.

Eliminar

Para eliminar un archivo, es necesario primero marcar el archivo indicado, y luego oprimir este botón. La modificación tendrá efecto definitivo una vez que se grabe .

Comentario

Permite agregar un comentario al archivo adjuntado. Es externo al adjunto y se muestra como un tooltip.

Para dar de alta o modificar un comentario, es necesario primero marcar el archivo indicado, y luego oprimir este botón. Se abrirá una pantalla en la cual se debe escribir los comentarios. Oprimiendo la tecla “ok” se graba las modificaciones realizadas.

Para visualizar el comentario, se debe situar el cursor sobre el archivo y aparecerá la descripción.

Modificar Archivos ya adjuntados

Se debe tener especial cuidado cuando se quiere realizar modificaciones a un archivo ya adjuntado. Para lograr esto, se debe primero guardar el archivo adjunto en forma local, es decir, en la PC desde la cual se esta trabajando, eliminar el adjunto del documento y grabar la modificación .

Realizar las modificaciones necesarias al archivo que se guardó en forma local, y una vez finalizadas éstas, adjuntarlo nuevamente como si fuese la primera vez.
