Bancos

El sistema permite la administración y el control del movimiento bancario mediante el registro de todos los tipos de transacciones que hacen a la operatoria bancaria. Esto es, la emisión de cheques, el ingreso de débitos y créditos bancarios, los pagos y cobranzas mediante bancos, las acreditaciones por liquidación de cupones de tarjetas. También será posible controlar las chequeras, efectuar conciliaciones bancarias, tener los saldos bancarios al día, etc.

A tal efecto la tabla Bancos permite registrar datos a los bancos con los que opera la empresa y, además, indicar un proveedor que va a definir el comportamiento impositivo a tener en cuenta cuando se registren movimientos que afecten las cuentas bancarias que para el banco se habiliten.

Se debe identificar el banco con un código y descripción y, luego, registrar la información que corresponda.

Código

Código identificatorio del banco.

Descripción

Texto que designa el nombre de la entidad bancaria.

Se exhibe junto al código cada vez que se invoca al banco.

Código del Banco Central

Código asignado por el banco central a la entidad bancaria que se está definiendo.

A tal efecto se incluye una tabla, llamada Códigos del Banco Central, con dicha codificación.

Proveedor

Código del proveedor que representa al banco. Este determinará el comportamiento impositivo del banco.
