Definición

La tarea denominada Definición de tablas de bonificaciones permite informar la tipificación de clientes y la tipificación de productos a tener en cuenta en el armado de las tablas de porcentajes. Ambos datos se suministran mediante atributos aplicables a clientes y a productos, respectivamente

Si se trabaja con múltiples empresas se deberán definir las tablas de bonificaciones de cada empresa con la que se trabaja. Las tablas de bonificaciones que se ingresen se registrarán en la empresa en la cual está el usuario en ese momento.

Los datos a ingresar son:

Empresa

Código de empresa en la cual se ingresa la tabla de bonificaciones que se está definiendo.

Este dato no se ingresa manualmente. Para las tablas de bonificaciones nuevas que se ingresen, se completa el dato automáticamente con el código de la empresa en la cual está el usuario en ese momento; y sólo podrán verse o modificarse las tablas de bonificaciones asociadas a esa empresa.

Código

Código identificatorio de la tabla de bonificación que se define.

Para que el sistema utilice una tabla de bonificación, el código de la misma debe informarse en las condiciones de venta que se asocien a los comprobantes que se ingresen en el sistema.

Descripción

Nombre asignado a la tabla de bonificación que se define.

Atributo Cliente

Código del atributo cuyos valores identifican a las diversas categorías de clientes que poseen bonificaciones diferenciadas.

Atributo Item

Código del atributo cuyos valores identifican a las diversas categorías de ítems que poseen bonificaciones diferenciadas.
